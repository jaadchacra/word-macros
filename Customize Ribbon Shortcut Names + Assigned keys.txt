Right Click the Ribbon -> Customize Ribbon -> Keyboard shortcuts: Customize -> All Commands

Collapse Heading
	Ctrl + Shift + E

Expand Heading
	Ctrl + E

Collapse ALL
	Ctrl + R

Expand ALL
	Ctrl + Shift + R
 
Increase Indent (like code)
	Ctrl + T

Decrease Indent (like code)
	Ctrl + Shift + T

CenterPara
	Ctrl + ~
 
TableInsertTable
	Ctrl + 1
 
Continue Title Numbering
	Alt + Enter

Case Conversion ALL
	Shift + f3
 
Headings
	Alt + 1 (2,3,4...)
	OR
	Ctrl + Shift + Alt (Q,A,S,Z,X,D,C) for headings 0,2,3,4,5,6,7 respectively

Font Colors
	ctrl + 4 (5,6,7,8,9,0)

Grow Font
	ctrl + ]   OR          ctrl + shift + >

Shrink Font
	ctrl + [   OR          ctrl + shift + <
 
Insert Annotation (Comment)
	ctrl + ;

Show Comments
	ctrl + \ (or ctrl + L)

ReadingMode
	Ctrl + ! (or ctrl + /)

ViewPage
	Escape button (or ctrl + shift + \ === ctrl + |)

NavPane
	ctrl + H
 
Close Window
	ctrl + W

Page Up
	ctrl + up

Page Down
	ctrl + down

HIGHLIGHTER
	ctrl + G

Background Colors
	ctrl + \   (on the right)


Macros
	In the list of shortcut categories, click on Macros (Instead of All Commands)
	Give each Macro the shortcut based on the comment within each VBA Function
